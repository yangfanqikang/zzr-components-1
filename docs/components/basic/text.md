# Text 文字

## 文字大小
| 层级  | 大小  |  描述  | 
| ----  | ----  |  ----  |
|  h1   | 32px  | 一级标题 |
|  h2   | 28px  | 二级标题 |
|  h3   | 24px  | 三级标题 |
|  h4   | 20px  | 四级标题 |
|  h5   | 18px  | 五级标题 |
|  h6   | 16px  | 六级标题 |
|  p    | 12/14  | 正文 |

## 文字行高
```
line-height: 1em;
line-height: 2em;
```
## 文字缩进
```
text-indent: 2em
```
## 文字对齐: 两端对齐,底端左对齐
```
text-align: justify;
text-align-last: left;
```
